# FUTURE_ML_02 — Support Ticket Classifier

## Task Overview
Automatically classify IT support tickets into categories
and assign priority levels using Machine Learning.

## Model Performance
- Algorithm: Logistic Regression
- Accuracy: 85.11%

## Categories Predicted
| Category | Precision |
|----------|-----------|
| Access | 92% |
| Administrative rights | 86% |
| HR Support | 85% |
| Hardware | 80% |
| Internal Project | 93% |
| Miscellaneous | 81% |
| Purchase | 97% |
| Storage | 93% |

## Priority System
| Priority | Condition |
|----------|-----------|
| Critical | Broken, down, urgent, crashed |
| High | Password expired, access blocked, storage full |
| Low | New requests, setup, installation |
| Medium | Everything else |

## Tools Used
- Python
- Scikit-learn
- NLTK
- Pandas
- Matplotlib & Seaborn

## Dataset
IT Service Ticket Classification Dataset — Kaggle
